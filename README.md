# 《倚天屠龙记》· 胡笳十八拍

> **以艺术点亮医学，守护女性健康美**
>
> 浙江大学《中西医学与美学》期末大作业
>
> 一款融合中医与西医女性健康知识的金庸武侠互动网页游戏

---

## 📖 项目简介

本项目以金庸经典武侠小说《倚天屠龙记》为叙事框架，以蔡文姬《胡笳十八拍》为精神内核，创作了一款交互式网页游戏。

玩家可选择女性角色（赵敏、周芷若），跟随她们的江湖旅程，经历18道关卡——每一关对应《胡笳十八拍》的一拍，每一关嵌入一个女性健康或中医学知识问题。通过选择题和简答题的交互形式，将专业医学知识转化为通俗易懂的游戏体验。

### 核心特色

- 🎮 **互动叙事**：基于原著的改编情节，选择不同角色体验不同故事线
- 🩺 **医学科普**：每关嵌入中西医女性健康知识（含中医经典、西医循证）
- 🤖 **AI对话**：简答题接入DeepSeek API，模拟医者与求医者的自然对话
- 📚 **学术严谨**：每道题附可展开/收起的参考文献（中英文，<5年权威文献）
- 🎨 **美学底蕴**：金庸武侠水墨风美术（含216+张GPT插图提示词）
- 🏮 **人文关怀**：结局以诗词结语，表达对天下太平、人间无病的愿景

## 🚀 快速开始

### 环境要求

- Node.js 18 或更高版本
- 现代浏览器（Chrome / Edge / Firefox）
- 如需简答题 AI 对话功能：DeepSeek API Key

### 运行方式

1. 将 `.env.example` 复制为 `.env`，填入新建的 DeepSeek API Key：
   ```bash
   DEEPSEEK_API_KEY=你的API密钥
   ```
2. 在项目根目录启动：
   ```bash
   npm start
   ```
3. 浏览器访问 `http://127.0.0.1:8080`。

请勿把 `.env` 上传、转发或截图。API Key 仅由本地服务端读取，不会发送到浏览器。

## 🎮 游戏玩法

### 角色选择
- **赵敏**（🌹朔漠玫瑰）：蒙古郡主路线 —— 从宫闱到江湖，18关涵盖全生命周期女性健康
- **周芷若**（🪷汉水白莲）：峨眉掌门路线 —— 从渔女到掌门，18关关注女性心理与生理健康

### 关卡结构
每关包含：
1. **场景建立**：5帧动态插图（使用AI生成，代码中已留占位符）
2. **情节对话**：基于原著改编的叙事文本
3. **医学问题**：
   - **选择题**（不可跳过）：4个选项，答对/答错均有详细解析
   - **简答题**（可跳过）：调用DeepSeek API进行多轮对话，用户可随时退出
4. **参考文献**：可展开/收起的学术引用

### 操作方式
- 鼠标点击：所有交互
- 空格键/回车：推进对话
- 参考文献面板：右下角📚按钮随时查阅
