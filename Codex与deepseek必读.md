3. 我认为博文、海报、视频等很难与其他同学竞争，因此我打算做一款交互式的小游戏，名为《倚天屠龙记》，完成科普任务、展现美学底蕴、彰显人文关怀。

## 游戏设计

### 游戏平台

游戏支持网页端体验。

### 游戏角色

游戏支持用户选择《倚天屠龙记》中的角色，多为女性角色，例如黄蓉、赵敏、周芷若、金花婆婆（黛绮丝）、梅超风等，也有男性角色，如郭靖、洪七公、黄药师等。

### 游戏动线

1. 用户选择角色后开始游戏。针对不同角色，设有不同的关卡系列。

2. 关卡的情节设计尽可能与原著呼应，例如时间、地点、人物、事件等。允许合理改写。

3. 在每个关卡中，用户会遇到关于女性健康或中医学的问题。

- 这些问题以不同形式发生，例如情节中某人生病、受伤、怀孕、衰老等，又如皇帝贴出皇榜、招人治病等，再如黄药师出题为难郭靖等，再如需要撰写或阅读药方等，再如需要识破敌人恶意药方或毒药等，再如作诗、对联盛会等，再如练功习武、养精蓄锐、颐养天年等。

- 这些问题以不同形式展示，例如旁白字幕，又如NPC台词字幕，再如皇榜、题壁诗等。这些问题主要以选择题的形式供用户回答，选择题不可跳过；较少以简答题的形式供用户回答，需要用户输入（一轮或多轮）回答，调用deepseek API接收用户回答并判断是否继续提问或给出提示，用户可随时跳过简答题，即使在对话中途。

- 我将deepseek API Key放在.env；对于每个问题的答案与参考文献，可供用户控制展开与收起。

4. 根据用户不同的作答，进入不同的情节分支。分支结局需体现美学底蕴与人文关怀，例如以原著中的一首诗结尾、表达叹惋或凭吊等，又如以毡房玩雕、大漠夕阳、轩窗画眉、架药生尘等场景结尾，表达对天下太平、国泰民安、海晏河清、人间无病的愿景，再如以战火连天、金戈铁马等场景结尾，表达忧国忧民、心系天下的人文关怀。
