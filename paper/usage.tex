\subsection*{代码包的移动规则}

老师拿到的是一个完整的项目文件夹。请把它看作一只装好了全部零件的箱子：下载、解压、复制或移动时，直接操作最外层文件夹即可，不要单独搬动里面的 \path{js/}、\path{css/}、\path{images/} 等文件夹。否则网页可能找不到图片、样式或程序。

项目已经上传到 GitHub。因为其中涉及医学知识和生成式模型回答，为避免尚未复核的内容被直接传播，仓库目前没有向公众开放。老师如需查看仓库、在线预览，或希望邀请其他教师参与复核，可通过钉钉或电话联系张天羽（19706101056）；我会协助开通仓库协作者或开发者权限。

\subsection*{本地运行}

如果只想阅读本文档，无须进行下面的操作。若想亲自体验小游戏，可按以下步骤进行：

\begin{enumerate}[leftmargin=*]
  \item 先安装 Node.js 18 或更高版本。如果不确定是否已经安装，可以直接联系我协助确认。
  \item 将收到的压缩包完整解压。打开解压后的最外层文件夹，不要进入某个内部子文件夹。
  \item 在文件夹空白处按住 \texttt{Shift} 并单击鼠标右键，选择在终端中打开；也可以在文件夹顶部地址栏输入 \texttt{powershell} 后按回车。
  \item 在出现的窗口中输入 \texttt{npm start}，然后按回车。看到服务已经启动的提示后，不要关闭这个窗口。
  \item 打开常用浏览器，在地址栏输入 \url{http://127.0.0.1:8080}，按回车即可进入游戏。
  \item 体验结束后，关闭刚才的终端窗口即可停止程序。
\end{enumerate}

剧情、图片、选择题、固定解析和参考资料不需要额外配置即可使用。简答题中的生成式对话需要本地私有配置；为保证访问凭据安全，本文档不展示也不传递任何密钥。若老师希望体验这部分，或在安装、打开终端、启动网页的任何一步遇到问题，请通过钉钉或电话联系张天羽：19706101056，我可以协助完成。
